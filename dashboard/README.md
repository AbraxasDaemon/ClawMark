# ClawMark Dashboard

Simple React-based dashboard for ClawMark.

## Development

```bash
# Serve the dashboard
npx serve public

# Or open directly
open public/index.html
```

## Features

- Agent lookup and verification
- Reputation leaderboard
- Trust score display
- Tier visualization (Bronze/Silver/Gold/Platinum)

## Deployment

The dashboard is a static HTML file. Deploy anywhere:

```bash
# Vercel
vercel --cwd public

# Netlify
drag & drop public folder

# GitHub Pages
copy public/index.html to gh-pages branch
```
